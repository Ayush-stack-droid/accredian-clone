# 🚀 Accredian Enterprise Clone

A responsive partial clone of the Accredian Enterprise website built using Next.js and Tailwind CSS.

## 🌐 Live Demo

👉 https://your-vercel-link.vercel.app

## 🛠️ Tech Stack

* Next.js (App Router)
* React (Functional Components + Hooks)
* Tailwind CSS
* Vercel (Deployment)

---

## ✨ Features

* Fully responsive (mobile + desktop)
* Smooth scrolling navigation
* Animated statistics section (count-up effect)
* Reusable component-based architecture
* Clean and modern UI
* Lead capture form with API integration
* Interactive FAQ accordion
* Testimonials section

## ⚙️ Setup Instructions

1. Clone the repository

git clone https://github.com/your-username/accredian-clone.git
cd accredian-clone

2. Install dependencies

npm install

3. Run development server

npm run dev

4. Open in browser

http://localhost:3000
